# Parking Attendant iOS App
This is an iOS app that allows parking attendant to take notes and give citation.


## Author
- Sy Le (https://www.linkedin.com/in/syle1021/)


## Stacks
### Front End (iOS)
- iOS 11
- AlomoFire (Network Requests)


### Web Server
- Node JS
- Express
- Sequlize
- Passport
- Sequelize
